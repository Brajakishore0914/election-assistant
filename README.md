# Election Process Assistant

A dynamic and interactive web application engineered for the **Hack2skill PromptWars Virtual Hackathon (Challenge 2)**. 

## Overview
This project provides a responsive, neon-themed user interface to educate users about the electoral process, timelines, and necessary steps. It features a custom chat assistant logic and a visual timeline component.

## Architecture & Tech Stack
* **Frontend:** HTML5, CSS3, Vanilla JavaScript
* **Containerization:** Docker
* **Web Server:** Nginx (Alpine)
* **Target Deployment:** Google Cloud Run

## Key Features
* Interactive chat interface for parsing and answering election-related queries.
* 3D perspective environment effects and modern neon aesthetic styling.
* Lightweight, fully containerized architecture designed for scalable cloud deployment.
